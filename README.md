# input-remapper

### Build guide

#### 1. Install SDL3
https://wiki.libsdl.org/SDL3/README-windows
tip (mingw) Install this MinGW package `mingw-w64-x86_64-toolchain` to install prereqs

#### 3. Set SDL3_DIR env variable to your SDL install dir


#### 4. Run `cmake --build .`


#### 5. `./mapper.exe`
